# Calendario raccolta rifiuti

Calendario bilingue per il sito **Tra il Verde e il Mare**. Mostra agli ospiti i giorni della raccolta differenziata, con eventi aggiornati tramite Google Calendar.

- Versione italiana e inglese.
- Vista mensile su desktop e lista su mobile.
- Indicazione dei sabati senza raccolta.
- Evidenziazione del giorno corrente.
- Integrazione nel sito tramite iframe con altezza automatica.
